# py-musicPlayer by @marurunk

### A music player for karaoke (Windows)

## Installation
1. install python3 in your Windows.

2. install pyglet:

```python 
    pip install pyglet
```

3. install colorama:

```python 
    pip install colorama
```
## Run instruction

1. run the program:

```python
    python main.py
```

2. Choose a srt file

3. Choose music file (compatible with pygame)
